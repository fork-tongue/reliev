# Installation

Reliev is published on [PyPI](https://pypi.org/project/reliev/):

```sh
pip install reliev
```

Or with [uv](https://docs.astral.sh/uv/):

```sh
uv add reliev
```

Reliev depends on [observ](https://github.com/fork-tongue/observ) and [patchdiff](https://github.com/fork-tongue/patchdiff). Python 3.13 or newer is required.
